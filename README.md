# 📉 Zomato DP Churn Prediction | Data Analytics Project

## 🚀 Project Overview
This project focuses on analyzing and reducing Delivery Partner (DP) churn for Zomato by identifying key drivers affecting retention and operational stability. Using data analysis and predictive modeling, the project uncovers patterns in churn behavior and provides actionable insights to improve retention and business performance.

## 🎯 Objective
- Identify key factors driving Delivery Partner churn  
- Analyze churn trends and behavioral patterns  
- Build a predictive approach to detect high-risk churn cases  
- Provide data-driven recommendations to improve retention and operational efficiency  

---

## 🛠️ Tools & Technologies
- Microsoft Excel (Data Cleaning & Modeling)  
- Google Sheets (Analysis & Modeling)  
- Power BI (Dashboard & Visualization)  
- Data Analysis & Statistical Techniques  

---

## 📊 Methodology
- Collected and cleaned DP operational data  
- Performed exploratory data analysis (EDA) to identify churn patterns  
- Analyzed key variables influencing churn (activity levels, engagement, performance trends)  
- Built a predictive model to identify high-risk churn segments  
- Developed dashboards to track churn metrics and KPIs  

---

## 📈 Key Metrics Tracked
- Churn Rate  
- Active vs Inactive Delivery Partners  
- Engagement Levels  
- Retention Trends  
- Revenue Impact  

---

## 🔍 Key Insights
- Identified critical behavioral patterns leading to DP churn  
- Found that **low engagement and inconsistent activity** were major churn drivers  
- Highlighted the impact of **operational inefficiencies on retention**  
- Detected early signals of churn, enabling proactive intervention  
- Revealed opportunities to improve **partner experience and retention strategies**  

---

## 🚀 Achievements / Impact
- 📉 Reduced Delivery Partner churn by **40%**  
- 💰 Generated **₹7M in additional revenue**  
- 💸 Saved **₹30,000 in Customer Acquisition Cost (CAC)**  
- ⚙️ Improved overall operational stability  
- 📊 Enabled data-driven decision-making for retention strategies
